| Layer          | Technology       | Why?                                    |
| -------------- | ---------------- | --------------------------------------- |
| Runtime        | Node.js          | Fast I/O and rich ecosystem             |
| Framework      | Express          | Simple and widely used                  |
| Language       | Javascript       | Type safety and maintainability         |
| Database       | PostgreSQL       | Relational data with strong consistency |
| ORM            | Prisma           | Excellent developer experience          |
| Cache          | Redis            | Fast caching and progress tracking      |
| AI             | Groq SDK         | Low latency inference                   |
| Workflow       | Temporal         | Durable background workflows            |
| Validation     | Zod              | Runtime validation                      |
| Authentication | JWT + bcrypt     | Stateless auth                          |
| Logging        | Winston/Pino     | Structured logs                         |
| Testing        | Jest + Supertest | Unit and integration testing            |
