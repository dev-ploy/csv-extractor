# CSV Extractor

AI Native CSV Importer that extracts CRM information from any valid CSV format using intelligent field mapping.

## Quick Start

```bash
# Backend (Node.js + Express + PostgreSQL)
cd backend
npm install
npm run dev

# Frontend (Next.js)
cd frontend
npm install
npm run dev
```
